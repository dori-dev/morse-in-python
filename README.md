# Morse in Python

Convert text(english) to morse codes and play morse sound!

#

# How to Play

## Installation Requirements

```
pip3 install python-vlc
```

## Run the Code

```
python3 main.py
```

#

# You Can Change

- You can change the sleep times and set_time_number in `functions.py` to change output sound!
